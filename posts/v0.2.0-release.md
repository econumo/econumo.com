# v0.2.0 Econumo release

Published: November 17, 2024
URL: https://econumo.com/posts/v0.2.0-release/

## Changelog

- Removed the legacy PostgreSQL dependency.
- Added an `X-Timezone` header to each API request to maintain consistent time zones between the frontend and backend.
- Addressed all of Rector's warnings.
- Removed the old analytics API.
- Update API documentation.
