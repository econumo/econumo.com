    Open Source

    Open Source #

    Econumo CE* consists of two parts: Frontend and Backend (API and Database).

    Backend (API) #

    Backend (API) is open-source under the AGPLv3 license and can be found on Github. It is written in PHP and uses the Symfony framework. The docker image is available on Docker Hub.

    Database #

    Econumo CE* uses SQLite as a database for simplicity of use.

    Frontend #

    Frontend is not open-source, but it is available for self-hosting. You can find the docker image on Docker Hub.
