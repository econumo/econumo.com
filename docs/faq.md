# Frequently Asked Questions

## Do you have plans to add automated import of transactions from banks?

Automated import of transactions from banks and other sources is not a priority at this point.
However, it might be added in the future as an optional feature.

As an alternative, you can use [CSV import](/docs/user-guide/csv-import-export/#exporting-your-transactions-to-csv).

If you have development skills and want to contribute, you are welcome to do so by using Econumo's API.
