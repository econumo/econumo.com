    Classifications #

    This section introduces the basics of ‘Classifications’ in Econumo, including Categories, Tags, and Payees. All three entities share similar functionalities:

    • Create a new item.
    • Update or delete an existing item.
    • Archive or unarchive any item.

    In this section #

    • Create a category, tag, or payee
    • Update or delete
    • Archive or unarchive

    Create a category, tag, or payee #

    To create a new item, go to Settings -> Categories / Payees / Tags and click on the Create button.

    You will need to fill out a few fields, such as:

    • Type (income/expense) - only for categories
    • Icon - only for categories
    • Name

    Update or delete #

    To update or delete an item, click on the three dots next to it and select the desired action, such as Edit or * Delete*.

    Deleting an item will remove it from all transactions and budgets.

    Archive or unarchive #

    You can archive items that are no longer relevant to you. Archived items won’t appear in the transactions dialog but will still be visible in the transactions list or a budget if you’ve budgeted them or had any associated spending.
