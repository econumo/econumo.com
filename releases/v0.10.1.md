# v0.10.1

Published: May 24, 2026
URL: https://econumo.com/releases/v0.10.1/

The **v0.10.1** release adds a version indicator so you can always tell which build of Econumo you're running, and fixes a startup issue on Alpine-based images.

## Changelog

- Added a display of Econumo's current version.
- Fixed an Alpine entrypoint failure by replacing `su` with `su-exec`.
