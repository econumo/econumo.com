# v1.0.2

Published: July 16, 2026
URL: https://econumo.com/releases/v1.0.2/

The **v1.0.2** release is a maintenance update with no user-facing changes. It bumps dependencies across the Go backend and the React frontend, and reworks the multi-arch release image build to cross-compile natively instead of running under QEMU.

## Changelog

- Dependency updates across the Go backend and the React frontend (crypto, mailer client, linting, test tooling, and more).
- Release image builds now cross-compile natively instead of emulating under QEMU, cutting multi-arch publish time from around 7.5 minutes to around 2.
- Upgraded the frontend build tooling to pnpm 11.
