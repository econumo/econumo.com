# v0.9.0

Published: January 20, 2026
URL: https://econumo.com/releases/v0.9.0/

The **v0.9.0** release brings Econumo together as a fully open-source project in a single repository, now licensed under the permissive MIT license.

It also improves the consistency of dialog windows and delivers an enhanced shared-access screen.

## Changelog
- Merged Econumo’s source code into a single repository.
- Changed the license from AGPLv3 to MIT.
- Improved dialog window consistency.
- Enhanced the shared access screen.
- Upgraded PHP to version 8.5.
