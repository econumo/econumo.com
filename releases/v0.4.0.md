# v0.4.0

Published: December 15, 2024
URL: https://econumo.com/releases/v0.4.0/

The **v0.4.0** release adds user registration and delivers a broad set of budgeting improvements and bug fixes.

## Changelog

- Added a registration feature and updated the configuration and login layout.
- Corrected the translation for "All accounts."
- Fixed budget calculations for individual users.
- Resolved issues with the modal window displaying the transaction list for budgets.
- Introduced user avatars in the transaction list modal for budgets.
- Fixed a bug that displayed ghost transactions when a user changed a transaction type.
- Improved the currency selection box in the Budget module.
- Refactored the structure of i18n labels.
- Enabled the ability to edit budgets on mobile devices.
- Fixed budget settings to ensure only accepted budgets can be selected.
- Ensured that the budgeted amount is consistently converted to the budget currency.
- Added a new System API for user creation: `/api/v1/system/create-user`.
